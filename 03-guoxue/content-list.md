## 弟子规选句（按季度）

### Q1（生活习惯）

```
父母呼，应勿缓
父母命，行勿懒
晨必盥，兼漱口
居有常，业无变
```

### Q2（待人）

```
兄道友，弟道恭
或饮食，或坐走
长者立，幼勿坐
借人物，及时还
```

### Q3–Q4（巩固 + 扩展）

```
对尊长，勿见能
路遇长，疾趋揖
凡是人，皆须爱
行高者，名自高
```

## 三字经选段

**Q2 起**，每周 2 句：

```
人之初，性本善。性相近，习相远。
苟不教，性乃迁。教之道，贵以专。
昔孟母，择邻处。子不学，断机杼。
养不教，父之过。教不严，师之惰。
```
