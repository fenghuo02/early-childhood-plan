# 资源与工具

> 以 **免费 / 低价、可重复** 为原则。链接以搜索关键词为准（避免失效）。

## 英语

| 类型 | 推荐 | 说明 |
|------|------|------|
| 视频 | YouTube 搜 **Super Simple Songs** | 儿歌首选，节奏慢 |
| 视频 | B站搜 **English Singsing** | 主题词汇动画 |
| 音频 | 网易云「英文儿歌精选」 | 洗澡 / 路上听 |
| 绘本 | 图书馆借 Eric Carle 系列 | 省预算 |
| App | **宝宝巴士英语**（适度） | 控制 10 min/天 |
| 自制 | 打印身体 / 颜色闪卡 | 见 `02-english/content-list.md` |

## 预算参考（一年）

| 项目 | 建议 spend |
|------|-----------|
| 新书 | 200–400 元 / 年（图书馆为主可更低） |
| 教具 | 100–200 元（磁力片 / 串珠） |
| App 会员 | 0–200 元（可不用） |
| **合计** | **300–800 元 / 年** 足够 |

## 不推荐（3 岁）

- 写字帖、算术题册
- 每天 1 小时屏幕英语课
- 以「考级」「认字量」为卖点的课程
