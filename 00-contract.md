## 我们的目标（勾选）

- [ ] 每天固定启蒙 **25–40 分钟**（分 3 块）
- [ ] 英语：以 **听 + 说 + 玩** 为主，不追求读写
- [ ] 国学：以 **古诗韵律 + 经典故事 + 好习惯** 为主，不追求全背
- [ ] 数学：以 **数感 + 形状 + 比较** 为主，不算式题
- [ ] 每周日复盘 5 分钟，每月复盘 15 分钟

## 家长承诺

我承诺：

1. **不吼不叫地完成陪伴**；今天状态差就缩短时间，不强行加时。
2. **不和别的孩子比进度**；只记录「比上周多坚持了一天」。
3. **连续 7 天中断**时，从最小任务重启（例如：只唱 1 首英文歌），而不是放弃整个计划。
4. **每月诚实填写** [`templates/monthly-review.md`](./templates/monthly-review.md)。

## 触发协议

| 情况 | 行动 |
|------|------|
| 连续 3 天没做 | 重读本契约，明天只做 10 分钟英语块 |
| 孩子明显抗拒某一块 | 停该块 3 天，换游戏形式再试 |
| 家长出差 / 生病 | 委托家人「维持最小版」：1 歌 + 1 诗 + 数 5 个苹果 |
| 想加新课程 / 买新教具 | 先完成当前季度计划满 1 个月再加 |
