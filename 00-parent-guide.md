# 家长操作手册

> 你是 3 岁孩子的**第一位老师**，不是「小学生的提前版教练」。

---

## 一、这份计划解决什么

| 问题 | 本方案的做法 |
|------|-------------|
| 不知道每天教什么 | 按 `05-daily-rhythm` + `06-year-plan` 照做 |
| 英语 / 国学 / 数学各买一堆课 | 统一节奏，每天各 8–15 分钟 |
| 教两天就放弃 | 契约 + 打卡 + 月度复盘 |
| 怕教错、怕超前 | `01-principles/development-at-3.md` 划边界 |

---

## 二、家长角色（只做这 4 件事）

1. **固定时间**：每天同一时段开始（误差 ±30 分钟可接受）
2. **陪玩式引导**：你演、你问、你夸，不考、不骂、不比较
3. **重复**：同一首儿歌 / 同一首诗，连做 5–7 天再换
4. **记录**：30 秒打勾 [`templates/daily-checklist.md`](./templates/daily-checklist.md)

**不要做的：**
- 不要要求 3 岁「认字写字」「背完整本三字经」
- 不要和邻居孩子比「我家都会 100 个单词了」
- 不要在情绪不好时强行上课（换 5 分钟亲子游戏再试）

---

## 三、环境准备（一次性，30 分钟）

### 物理角落「启蒙小站」

- 小桌或地垫 + 固定位置
- 一个收纳盒：闪卡、绘本 5–10 本、积木 / 磁力片
- 墙上可贴：本周古诗、数字 1–10、颜色卡

### 数字资源（见 `07-resources/links-and-tools.md`）

- 英语：1 个儿歌播放列表 + 2–3 本绘本
- 国学：1 个古诗音频 + 1 本带拼音的启蒙书
- 数学：家里现有积木、筷子、水果即可

---

## 四、每日流程（25–40 分钟）

```
① 英语块  10–15 min  →  02-english/
② 国学块  8–12 min   →  03-guoxue/
③ 数学块  8–12 min   →  04-math/
④ 打勾     30 sec     →  templates/daily-checklist.md
```

**顺序可调**，但建议「先动后静」：数学（动手）→ 英语（听唱）→ 国学（朗读）→ 睡前故事。

---

## 五、遇到这些情况怎么办

| 情况 | 做法 |
|------|------|
| 孩子不肯坐 | 缩短到 5 分钟；换站立游戏（TPR） |
| 只会听不说 | 正常；3 岁大量「沉默期」，继续输入 |
| 今天特别棒 | 多夸「你今天很专注」，不加量 |
| 连续 3 天没做 | 读 `00-contract.md`，从「只唱 1 首歌」重启 |
| 老人说「太小了」 | 给看 `01-principles/development-at-3.md` 边界说明 |

---

## 六、与 4 岁的衔接

本计划按 **12 个月（3 岁→4 岁）** 设计。第 10–12 月会逐步：
- 英语：从单词到简单句型（I like… / This is…）
- 国学：从听背到讲大意
- 数学：从 1–5 到 1–10，引入简单加减故事
